# PLANE WAR Game Based on Q-Learning
This project is a 2D airplane shooting game based on Q-Learning and Pygame.
## Project Files
- Assignment1 - ZHANG Lin.py: Complete game code with Q-table save/load
- Image resources: player.png, enemy_small.png, enemy_large.png
## How to Run
1. Install pygame: pip install pygame
2. Run the py file directly
3. Select AI/Human mode in the start menu
